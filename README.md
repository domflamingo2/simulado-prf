## 🧭 Como usar (PASSO A PASSO)

### 1️⃣ Instale as dependências

```bash
npm install
```

---

### 2️⃣ Rode o projeto

```bash
npm run dev
```

---

### 3️⃣ Acesse

```
http://localhost:3000
```

### 4️⃣ Inicie um simulado

> prática → análise → melhoria
